# Security Policy

## Backend Status
The Volt Backend is currently in an early MVP stage. It provides a metadata storage layer for the Volt protocol.

## Security Considerations
- **No Authentication**: This MVP release does not include authentication or authorization. All endpoints are public.
- **Data Integrity**: Users should verify on-chain data against the metadata provided by this service.
- **Production Use**: We do not recommend using this service in production environments without additional security layers (WAF, Auth, Rate Limiting).

## Reporting Vulnerabilities
Please report any security vulnerabilities to:

**Email:** security@volt-protocol.example.com

## Responsible Disclosure
We ask that you:
- Provide us reasonable time to investigate and fix issues before public disclosure.
- Do not exploit vulnerabilities for personal gain.
- Provide clear technical details for reproduction.

Thank you for helping secure the Volt protocol.
